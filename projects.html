Projects

Pixel Engine - GitHub

A library for Java designed to provide the simplest possible interface for drawing pixels to the screen. It's intended to be a learning tool to help people new to graphics programming become more comfortable with the logic behind drawing complex shapes on a 2D plane.

Kana Strokes - GitHub

An hiragana and katakana stroke order memorization app for Android. It uses a spaced repetition system similar to WaniKani to attempt to optimize learning.

Game Boy Emulator - GitHub

A Game Boy emulator programmed in C++.
